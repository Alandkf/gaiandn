This project is a web application designed to manage driver information efficiently. It utilizes a combination of technologies including Node.js, Express, Sequelize, and EJS for server-side rendering. The application features a responsive design with dynamic sidebar behavior, ensuring optimal usability across various screen sizes. Key functionalities include creating, updating, and deleting driver records, with soft delete functionality implemented to preserve data integrity. The frontend is enhanced with JavaScript to provide a seamless user experience, including real-time search and responsive sidebar toggling.
